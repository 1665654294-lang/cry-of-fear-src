# Cry of Fear Workspace

This is a writable copy of `external/hlsdk-portable` for the Cry of Fear recreation work.

Build it from the repository root:

```bat
build_game.bat
```

The default output is installed to `out\xash3d\cryoffear`:

- `cl_dlls\client.dll`
- `cl_dlls\hl.dll`

Keep upstream reference code in `external/hlsdk-portable`; make game-specific changes here.
